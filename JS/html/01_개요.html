자바스크립트 개요

        Javascript
         - html의 조작과 변경을 담당하는 언어
         - html로 웹 문서의 틀을 만들고, css로 디자인을 정의하며,js로 웹페이지상의 이벤트가
           발생했을때 실행할 동작을 정의한다.
         - 웹 브라우저에서 실행 가능한 인터프리터 방식의 스크립트 언어
           *인터프리터 : 컴파일 업싱 한줄씩 실행되는 언어

        장점
         - 코드를 작성한 후 바로 실행결과를 볼 수 있기 때문에 간단한 개발에 용이하다.
         - 클래스, 접근제한자, 반환형등 고급 개념이 없이 사용가능하기 때문에 초보자가 다루기 쉽다.
         - 웹브라우저의 기본 스크립트 언어로 범용성이 넓다.

        단점
         - 컴파일 방식의 언어보다 실행속도가 느리다.
         - 큰규모의 어플리케이션 개발에 있어서"타입 안전성"이 부족하다.

1.자바스크립트 사용법

  1. illine방식: 이벤트를 부여하고자 하는 요소 내에서 간단한 소스코드를 작성해서 실행시키는 방법
  2. internal 방식 : 현재문서 내부에서 script 태그 영역에 소스코드를 작성하여 실행하는 방법
  3. external 방식 : 현재문서 외부에서 별도의 js파일을 생성한 후 가져다가 실행하는 방식
