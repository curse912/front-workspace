그 밖의 글자 관련 문장 태그들

strong : 글자를 굵게 표시하는 태그

b : 글자를 굵게(bold)표시하는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그

mark : 형광펜효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

ins : 삽입된 글자 태그는 밑줄이 그어진다.

strike : 글자에 취소선이 그어지는 태그

del : 삭제된 글자 태그는 취소선이 그어진다.

small : 글자를 작게 표시하는 태그 하이

big : 글자를 크게 표시하는 태그 하이

IOT 란 각종 사물에 센서와 통신기능을 탑재해 인터넷에 연결하는 기술을 의미한다.

sub : 기본글자에 아래첨자를 나타내는 태그

sup : 기본글자에 위첨자를 나타내는 태그

code : 코드조각을 나타내는 태그 System.out.println("Hello, World!");

글자관련 태그 응용1

태그들은 해당 태그 내에서 중첩으로 사용 가능하다. 굵은 글자를 사용할 수 도 있고, 기울이거나, 취소선을 넣는 등 다양하게 사용할 수 있다.
형광펜을 넣어 글자를 강조할 수 있다. 정말 중요한 내용의 경우 태그들을 여러개 중첩하기도 한다.
