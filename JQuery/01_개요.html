jQuery 개요

jQuery란

    기존에 자바스크립트 언어 만으로 구현하기에 복잡했던 구문 작성들을 간소화 하기 위해 개발된 javascript기반 라이브러리.
    즉, javascript 방식으로 작성된 유용한 함수 및 매서드들이 정의 되어있다.
    
    복잡한 자바스크립트 구무을 보다 짧고 쉽게 개발할 수 잇어 생산성이 크게 향상된다.

    DOM요소와 관련된 조작기능이나, 이벤트 처리기능, AJAX기능, 추가 확장 플러그인등 
    다양한 기능들을 제공한다.

jQuery 라이브러리 연결 방법

head태그 내부에 cript태그로 jQuery관련 .js파일 가져오기
1. jQuery라이브러리를 사이트에서 직접 다운로드 받은 후 경로를 지정하는 오프라인 방식
2. CDN(Content Delivery Network)를 통한 다운로드 방식(온랑니 방식)
- jQuery 라이브러리를 제공하는 사이트 url을 src연결경로로 지정

jQuery 구문 작성

jQuery 구문으로 DOM요소를 다루는 구문을 작성할 예정이면, jQuery명령어들은 문서상의 모든 html요소들이 로드 된 후에 실행 되어야한다.

- 순수 자바스크립트로 html페이지가 모두 로드된 이후에 코드가 실행되게 하려면 


    window.onload = function(){
        //실행 내용
    }
- jquery에서의 onload함수 

    1) jQuery(document).ready(function(){
        실행내용;
    });

    2) $(document).ready(function(){
        실행 내용;
    });

    3) $(function(){
        실행내용;
    });
